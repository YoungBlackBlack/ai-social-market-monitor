更新时间：2026-06-02 13:45 CST

## 当前状态

本项目是一个基于 Exa Search API 的 AI 社交、AI 泛娱乐、语言交换和线下真人社交研究网页。当前主交付物是本地网页和单文件 HTML 报告，CSV/Markdown/JSON 用于复核和归档。

## 交付收口

当前版本可作为对外展示版使用。网页已把首页结论、竞品矩阵、语言相似度、融资/新闻、IRL 线下真人社交、Monitor、证据审计和 Playbook 拆分到独立页面；CSV/JSON/Markdown/时间戳快照均已保存。严格口径仍保持 `not-mathematically-complete`：这不是数据未完成，而是因为实时消费社交市场不能被数学穷尽；当前硬门槛为阻塞型候选 0、latest alert 待复核 0、recent signal 待复核 0。

## 后续维护规则

- 每次新增功能或导出，更新本文件的“功能清单”和“版本记录”。
- 每次 monitor 发现新增 URL，先进入 `latest-alert-review.csv`，确认后再升级主表、融资时间线或背景信号。
- 每次 full rebuild 发现新候选，先进入 source linkage / coverage gap / candidate deep-dive，不直接声称已穷尽。
